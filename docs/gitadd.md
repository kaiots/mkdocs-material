# Установка GIT
## Виртуальная среда 

Создать:
``` bash
python3 -m venv venv
```

Активация:
``` bash
source venv/bin/activate
```
Зафиксировать изменения git:
``` bash
git add .  
```
Добавить коментарий:
``` bash
git commit -m $'Adding inital document file54555' 
```
Отправить в ветку main:
``` bash
git push origin main
```
